# rj

`rj` is a command line tool show the HTTP Response as JSON

![Screenshot](https://user-images.githubusercontent.com/10682/140741605-7ae52cdf-e5c3-4c72-9cb7-78e7dbc7c3ea.png)


## Installation

```
$ go get github.com/yusukebe/rj/cmd/rj
```

## Usage

```
$ rj http://example.com/
```
